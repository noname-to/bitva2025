Репозиторий с исходным кодом команды Безымянные для хакатона Битва тИТанов 2025 (РУТ МИИТ)

Кейс: МАКСИМИЗАЦИЯ ДОХОДНОСТИ АВИАКОМПАНИИ ЧЕРЕЗ ОПТИМИЗАЦИЮ РАСПИСАНИЯ (АФЛК-Системы, Аэрофлот)

Валидация решения:
validation.csv - исходный валидационный датасет от заказчика
validation-predicted.csv - итоговый датасет с предсказанным доходом и бронированиями
challenge.ipynb - пайплайн вышеописанного предсказания

Внешние датасеты:
airports.csv - сведения об аэропортах
geonames-all-cities-with-a-population-1000.csv - сведения о городах

Внутренние датасеты:
frontend.json - данные под фронтенд
frontend.pkl - данные под фронтенд
prepared-dataset.csv - датасет после предобработки и обогащения
prepared-dataset.pkl - датасет после предобработки и обогащения
hackathon_data_main.csv - исходные данные от заказчика (10 месяцев)

Ноутбуки:
analysis.ipynb - временные графики (анализ данных)
compare.ipynb - временные графики (оценка эффекта)
preparation.ipynb - предобработка и обогащение
regressorv2.ipynb - регрессор (обучение модели)
schedulerv2.ipynb - планировщик расписания

Обученные модели:
models/full-context-load.cbm - полноконтекстная модель для прогноза бронирований
models/full-context-revenue.cbm - полноконтекстная модель для прогноза дохода
models/small-context-load.cbm - ограниченная модель для прогноза бронирований
models/small-context-revenue.cbm - ограниченная модель для прогноза дохода
